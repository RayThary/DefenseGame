# Little Wars / Defense Game

Unity 기반 모바일 디펜스 개인 프로젝트입니다.

## 프로젝트 개요
- 장르: 디펜스 게임
- 개발 엔진: Unity 2022.3.4f1
- 개발 기간: 약 3주
- 목표: 유닛을 구매하고 소환하여 상대 기지를 공격하는 라인 디펜스 구조 구현

## 주요 구현
- 유닛 구매 및 소환 시스템
- 유닛 타입별 공격 패턴 구현
- 적 AI 및 적 유닛 자동 소환
- 난이도별 클리어 표시
- PlayerPrefs 기반 클리어 기록 저장
- 사운드 매니저 및 SFX 풀링
- 버튼 기반 카메라 이동

## 기술 포인트
- 유닛의 진영과 종류를 Enum으로 구분하여 타입별 패턴 적용
- 전사, 궁수, 마법사, 기병 등 유닛별 공격 방식 구현
- 버튼 클릭 시 현재 코스트를 확인하고 조건을 만족하면 유닛 구매 처리
- 적은 시간에 따라 자원 회복량과 소환 가능 범위가 증가하도록 구성
- 난이도 클리어 여부를 PlayerPrefs로 저장하고 메인 화면에서 메달 표시
- SFX는 풀링 매니저를 활용하여 오디오 소스를 재사용하도록 구성

## Note
일부 그래픽 리소스는 외부 에셋을 사용했습니다. 본 프로젝트는 학습 및 포트폴리오 목적의 개인 프로젝트이며, 주요 구현 범위는 유닛 구매, 타입별 전투 로직, 적 AI, 클리어 저장, 사운드 매니저 구조입니다.
